## Part 1: 核心理念与协作框架

### 1.1 核心理念：AI 赋能，人工主导

鉴于抖音平台对境外 IP 的严格风控限制，我们采用一种更高效、更安全的“人机协作”模式：

- **AI 团队 (Manus)**：作为您的“超级大脑”，负责所有**策略制定、内容创意、数据分析**等脑力密集型工作。产出可直接复制粘贴的标准化内容和决策建议。
- **人工执行 (您或您的团队)**：作为“唯一执行者”，负责所有**账号登录、内容发布、与平台交互**等实际操作。只需“复制-粘贴-点击”，即可完成高质量的运营动作。

这种模式既能发挥 AI 在内容生产和数据处理上的强大优势，又能确保账号 100% 安全合规。

### 1.2 协作分工与标准作业流程 (SOP)

| 运营环节 | AI 团队 (Manus) 负责 | 人工执行 (您) 负责 |
| :--- | :--- | :--- |
| **1. 选品策略** | - 分析全网家居好物趋势<br>- 输出《每周选品决策报告》 | - **决策**：从报告中最终敲定本周主推款 |
| **2. 内容创作** | - 围绕主推款生成多版本视频脚本<br>- 提供爆款标题和标签组合 | - **执行**：拍摄/剪辑视频<br>- **发布**：将脚本内容复制到抖音发布页 |
| **3. 数据监控** | - 提供数据分析框架和指令模板 | - **执行**：每日从抖音后台导出数据<br>- **提问**：将数据粘贴给 AI，获取分析报告 |
| **4. 社群互动** | - 生成评论区、私信的高频问题标准回复话术库 | - **执行**：将话术复制粘贴，回复用户 |
| **5. 供应链管理** | - 提供供应商筛选标准和沟通模板 | - **执行**：与供应商实际沟通和下单 |
